Accessibility features

1. All buttons can be accessed with the Tab Key.
2. Each tab of this dialer as well as all buttons in the dialer tab has a accesskey and thus a keyboard shortcut.
3. An accessible table in the contacts tab as well as for the keyboard shortcuts here.
4. Basic ARIA landmark roles and a ARIA live region.
